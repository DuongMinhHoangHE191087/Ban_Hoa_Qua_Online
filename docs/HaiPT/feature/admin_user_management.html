Iteration 1

📖 Chức năng 7: Admin User Management – Quản trị người dùng & Phê duyệt Shop

Bảng điều khiển dành cho Admin để quản lý danh sách tài khoản khách hàng, chủ cửa hàng, khóa tài khoản vi phạm và phê duyệt hồ sơ mở shop.

Luồng Hoạt Động (Data Flow)
1
Vào trang quản trị
Admin truy cập đường dẫn /admin/users hoặc /admin/shops.
▼
2
Truy vấn danh sách
UserManageServlet và AdminShopManageServlet lấy dữ liệu người dùng/hồ sơ shop hiển thị lên bảng quản trị.
▼
3
Khóa tài khoản
Admin bấm Khóa tài khoản -> AJAX gửi tới AdminUserStatusAPI để cập nhật trạng thái user thành LOCKED trong DB.
▼
4
Thu hồi phiên làm việc
API gọi tiếp AdminUserRevokeSessionsAPI để xóa sạch token trong user_sessions, ép tài khoản đó đăng xuất ngay lập tức.
▼
5
Phê duyệt mở shop
Admin xem hồ sơ đăng ký tại ShopApprovalServlet, nhấn Phê duyệt -> ShopApprovalAPI cập nhật trạng thái shop thành APPROVED và đổi role của user thành SHOP_OWNER.
Các Thành Phần & Lớp Liên Quan
Tên lớp / Tên bảng Loại thành phần Nhiệm vụ trong chức năng
UserManageServlet Servlet Quản lý hiển thị danh sách người dùng phân trang cho Admin.
AdminUserStatusAPI Servlet API nhận yêu cầu khóa/mở khóa tài khoản người dùng.
AdminUserRevokeSessionsAPI Servlet API thực hiện xóa toàn bộ token phiên đăng nhập của người dùng bị khóa.
ShopApprovalServlet Servlet Xem chi tiết tài liệu chứng minh và phê duyệt đơn mở cửa hàng.
ShopApprovalAPI Servlet API xử lý duyệt hoặc từ chối yêu cầu đăng ký mở shop.
UserService Service Cung cấp nghiệp vụ khóa tài khoản và đồng bộ đăng xuất.
ShopService Service Xử lý nghiệp vụ duyệt shop và tự động nâng cấp vai trò người dùng.
UserDAO DAO Thực hiện truy vấn, lọc dữ liệu tài khoản và cập nhật cờ trạng thái.
ShopProfileDAO DAO Cập nhật trạng thái phê duyệt của cửa hàng.
user-list.jsp JSP Giao diện quản lý người dùng của Admin.
shop-approvals.jsp JSP Giao diện hiển thị danh sách yêu cầu mở shop đang chờ phê duyệt.
