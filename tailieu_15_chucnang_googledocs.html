Chức năng 1: Thêm mới sản phẩm (Có upload ảnh & lưu DB)

1. Thông tin chung

  • Tên chức năng: Thêm mới sản phẩm và tài liệu xác nhận nguồn gốc.
  • Đối tượng sử dụng (Actor): Chủ cửa hàng (Shop Owner).
  • Mục tiêu: Đăng bán sản phẩm hoa quả mới kèm các biến thể bán hàng, tùy chọn bao bì đóng gói, tài liệu xác nhận nông sản sạch và hình ảnh mô tả.
[BẢO MẬT & PHÂN QUYỀN]: Hệ thống bắt buộc kiểm tra xem người dùng hiện tại đã đăng nhập chưa và có vai trò là SHOP_OWNER hay không. Nếu không thỏa mãn, Servlet lập tức kết thúc xử lý và chuyển hướng (redirect) người dùng về trang đăng nhập /auth/login để tránh việc truy cập trái phép.

2. Luồng hoạt động chi tiết

  • Bước 1 (Client - Trình duyệt): Chủ shop điền thông tin sản phẩm trên giao diện, thêm các biến thể phân loại (SKU, giá bán, trọng lượng), tải lên tệp tài liệu xác thực và các ảnh sản phẩm. Bấm nút "Lưu sản phẩm" (Request gửi đi dưới dạng multipart/form-data).
  • Bước 2 (Servlet - ProductCreateServlet):
    • Thực hiện kiểm tra quyền đăng nhập và vai trò SHOP_OWNER của Session hiện tại. (Xem code Java nhãn [Bước 2 - Kiểm tra đăng nhập và vai trò])
    • Đọc các trường thông tin sản phẩm và danh sách các biến thể phân loại, tùy chọn đóng gói gửi lên từ form. (Xem code Java nhãn [Bước 2 - Đọc dữ liệu từ form request])
    • Thực hiện validate dữ liệu đầu vào phía Server (kiểm tra rỗng, đúng định dạng số, giới hạn hạn sử dụng). (Xem code Java nhãn [Bước 2 - Validate dữ liệu phía Server])
    • Đọc các file đính kèm và kiểm tra tính hợp lệ định dạng. (Xem code Java nhãn [Bước 2 - Đọc và validate file đính kèm])
  • Bước 3 (Service/Util): Gọi FileUploadUtil ghi các file thực tế vào ổ đĩa trên máy chủ và trả về đường dẫn tương đối (relative path). (Xem code Java nhãn [Bước 3 - Ghi file vật lý lên server])
  • Bước 4 (DAO Layer - Bắt đầu Transaction):
    • Gọi ProductDAO.save() chèn thông tin chung vào bảng products và lấy ra product_id tự động sinh. (Xem code Java nhãn [Bước 4 - Lưu thông tin sản phẩm chung])
    • Gọi ProductImageDAO.save() cho từng file ảnh để lưu vào bảng product_images (ảnh đầu tiên là ảnh chính). (Xem code Java nhãn [Bước 4 - Lưu danh sách hình ảnh])
    • Gọi ProductVariantDAO.save() để tạo các biến thể trong bảng product_variants. (Xem code Java nhãn [Bước 4 - Lưu danh sách phân loại biến thể])
    • Gọi ProductPackagingOptionDAO.save() để lưu các tùy chọn đóng gói đi kèm. (Xem code Java nhãn [Bước 4 - Lưu danh sách tùy chọn đóng gói])
  • Bước 5 (Database & Response): Ghi nhận dữ liệu thành công, redirect về trang danh sách sản phẩm và hiển thị thông điệp thành công. (Xem code Java nhãn [Bước 5 - Trả về phản hồi và redirect])

// ProductCreateServlet.java - Phương thức doPost() xử lý luồng thêm mới sản phẩm hoàn chỉnh
@Override
protected void doPost(HttpServletRequest req, HttpServletResponse resp)
        throws ServletException, IOException {
    req.setCharacterEncoding("UTF-8");
    resp.setContentType("text/html;charset=UTF-8");
    resp.setCharacterEncoding("UTF-8");

    // [Bước 2 - Kiểm tra đăng nhập và vai trò]: Xác thực Session của người dùng hiện hành
    HttpSession session = req.getSession();
    User currentUser = SessionUtil.getCurrentUser(session);
    if (currentUser == null || !AppConfig.ROLE_SHOP_OWNER.equals(currentUser.getRole())) {
        resp.sendRedirect(req.getContextPath() + "/auth/login");
        return;
    }

    // [Bước 2 - Đọc dữ liệu từ form request]: Đọc các thông tin văn bản và mảng phân loại
    String name = req.getParameter("name");
    String description = req.getParameter("description");
    String originCountry = req.getParameter("originCountry");
    String originRegion = req.getParameter("originRegion");
    String harvestDateStr = req.getParameter("harvestDate");
    String shelfLifeStr = req.getParameter("shelfLifeDays");
    String storageInstruction = req.getParameter("storageInstruction");
    String categoryIdStr = req.getParameter("categoryId");
    boolean isOrganic = req.getParameter("isOrganic") != null;
    boolean isImported = req.getParameter("isImported") != null;
    
    String[] variantLabels = req.getParameterValues("variantLabel");
    String[] variantPrices = req.getParameterValues("variantPrice");
    String[] variantWeights = req.getParameterValues("variantWeight");
    String[] variantDiscountPrices = req.getParameterValues("variantDiscountPrice");
    String[] variantDiscountStarts = req.getParameterValues("variantDiscountStart");
    String[] variantDiscountEnds = req.getParameterValues("variantDiscountEnd");
    
    String[] packagingLabels = req.getParameterValues("packagingLabel");
    String[] packagingPriceAdds = req.getParameterValues("packagingPriceAdd");

    List errors = new ArrayList<>();
    // [Bước 2 - Validate dữ liệu phía Server]: Kiểm tra tính hợp lệ của dữ liệu trước khi lưu
    if (name == null || name.trim().isEmpty()) {
        errors.add("Tên sản phẩm không được để trống.");
    }
    int categoryId = 0;
    try {
        categoryId = Integer.parseInt(categoryIdStr);
    } catch (NumberFormatException e) {
        errors.add("Vui lòng chọn danh mục hợp lệ.");
    }
    if (variantLabels == null || variantLabels.length == 0) {
        errors.add("Sản phẩm phải có ít nhất một phân loại/biến thể.");
    }

    // [Bước 2 - Đọc và validate file đính kèm]: Đọc các file ảnh và tài liệu đính kèm
    List imageParts = new ArrayList<>();
    Part verificationDocPart = null;
    try {
        verificationDocPart = req.getPart("verificationDoc");
        if (verificationDocPart == null || verificationDocPart.getSize() == 0) {
            errors.add("Vui lòng tải lên giấy tờ xác nhận nguồn gốc nông sản.");
        }
        for (Part part : req.getParts()) {
            if ("images".equals(part.getName()) && part.getSize() > 0) {
                imageParts.add(part);
            }
        }
    } catch (Exception e) {
        errors.add("Lỗi đọc tệp tải lên: " + e.getMessage());
    }

    if (!errors.isEmpty()) {
        SessionUtil.flashError(session, String.join("
", errors));
        resp.sendRedirect(req.getContextPath() + "/shop/products");
        return;
    }

    try {
        String uploadDir = getServletContext().getRealPath("");
        // [Bước 3 - Ghi file vật lý lên server]: Lưu giấy tờ chứng nhận và trả về đường dẫn tương đối
        String docPath = FileUploadUtil.saveShopDoc(verificationDocPart, uploadDir, currentUser.getUserId());

        Product p = new Product();
        p.setOwnerId(currentUser.getUserId());
        p.setCategoryId(categoryId);
        p.setName(name.trim());
        p.setDescription(description != null ? description.trim() : null);
        p.setOriginCountry(originCountry != null ? originCountry.trim() : null);
        p.setOriginRegion(originRegion != null ? originRegion.trim() : null);
        p.setHarvestDate(harvestDateStr != null && !harvestDateStr.trim().isEmpty() ? LocalDate.parse(harvestDateStr) : null);
        p.setShelfLifeDays(shelfLifeStr != null && !shelfLifeStr.trim().isEmpty() ? Integer.parseInt(shelfLifeStr) : null);
        p.setStorageInstruction(storageInstruction != null ? storageInstruction.trim() : null);
        p.setStatus("ACTIVE");
        p.setIsOrganic(isOrganic);
        p.setIsImported(isImported);
        p.setVerificationDocPath(docPath);
        
        String autoApproveVal = systemConfigDAO.getValue("product_auto_approve");
        p.setApprovalStatus("true".equalsIgnoreCase(autoApproveVal) ? "APPROVED" : "PENDING");

        // [Bước 4 - Lưu thông tin sản phẩm chung]: Lưu vào bảng products
        int productId = productDAO.save(p);

        // [Bước 4 - Lưu danh sách hình ảnh]: Gọi DAO lưu thông tin file ảnh tương ứng sản phẩm
        int imageIndex = 0;
        for (Part part : imageParts) {
            String relativePath = FileUploadUtil.save(part, uploadDir);
            if (relativePath != null) {
                ProductImage img = new ProductImage();
                img.setProductId(productId);
                img.setFilePath(relativePath);
                img.setDisplayOrder(imageIndex);
                img.setIsPrimary(imageIndex == 0);
                productImageDAO.save(img);
                imageIndex++;
            }
        }

        // [Bước 4 - Lưu danh sách phân loại biến thể]: Lưu các phân loại biến thể sản phẩm
        if (variantLabels != null) {
            for (int i = 0; i < variantLabels.length; i++) {
                ProductVariant variant = new ProductVariant();
                variant.setProductId(productId);
                variant.setSku("SP-" + productId + "-" + (i + 1));
                variant.setVariantLabel(variantLabels[i].trim());
                variant.setPrice(new BigDecimal(variantPrices[i].trim()));
                variant.setStockQuantity(0); 
                variant.setWeightKg(new BigDecimal(variantWeights[i].trim()));
                variant.setIsActive(true);
                productVariantDAO.save(variant);
            }
        }

        // [Bước 4 - Lưu danh sách tùy chọn đóng gói]: Lưu các tùy chọn đóng gói đi kèm
        if (packagingLabels != null) {
            dao.catalog.ProductPackagingOptionDAO ppoDAO = new dao.catalog.ProductPackagingOptionDAO();
            for (int i = 0; i < packagingLabels.length; i++) {
                String pLabel = packagingLabels[i];
                if (pLabel != null && !pLabel.trim().isEmpty()) {
                    model.entity.catalog.ProductPackagingOption option = new model.entity.catalog.ProductPackagingOption();
                    option.setProductId(productId);
                    option.setLabel(pLabel.trim());
                    option.setPriceAdd(new BigDecimal(packagingPriceAdds[i].trim()));
                    option.setIsActive(true);
                    ppoDAO.save(option);
                }
            }
        }

        // [Bước 5 - Trả về phản hồi và redirect]: Chuyển hướng người dùng và lưu flash message thành công
        SessionUtil.flashSuccess(session, "Thêm sản phẩm mới thành công!");
        resp.sendRedirect(req.getContextPath() + "/shop/products");

    } catch (SQLException e) {
        SessionUtil.flashError(session, "Lỗi cơ sở dữ liệu khi thêm mới sản phẩm: " + e.getMessage());
        resp.sendRedirect(req.getContextPath() + "/shop/products");
    }
}

Chức năng 2: Chỉnh sửa thông tin sản phẩm và cập nhật DB

1. Thông tin chung

  • Tên chức năng: Sửa thông tin sản phẩm và cập nhật dữ liệu.
  • Đối tượng sử dụng (Actor): Chủ cửa hàng (Shop Owner).
  • Mục tiêu: Thay đổi thông tin sản phẩm hiện tại, cập nhật giá bán, cân nặng của các phân loại biến thể.
[BẢO MẬT & PHÂN QUYỀN]: Hệ thống xác thực đăng nhập của Shop Owner và kiểm tra quyền sở hữu đối với sản phẩm này bằng cách đối chiếu owner_id của sản phẩm trong Database với ID của người dùng trong Session. Nếu không trùng khớp, từ chối xử lý và chuyển hướng về trang đăng nhập hoặc báo lỗi 403 Forbidden.

2. Luồng hoạt động chi tiết

  • Bước 1 (Client): Chủ shop bấm nút "Sửa" trên giao diện danh sách. Hệ thống gửi yêu cầu GET lên /shop/product-edit?productId=X. Giao diện hiển thị form chứa toàn bộ thông tin sản phẩm cũ được lấy ra từ Database.
  • Bước 2 (Servlet - ProductEditServlet):
    • Xác thực vai trò Shop Owner và quyền sở hữu sản phẩm hiện hành. (Xem code Java nhãn [Bước 2 - Auth & Owner Check])
    • Đọc các tham số mới thay đổi từ form gửi lên (POST). (Xem code Java nhãn [Bước 2 - Đọc tham số cập nhật])
    • Validate tính hợp lệ của dữ liệu đầu vào. (Xem code Java nhãn [Bước 2 - Validate dữ liệu cập nhật])
  • Bước 3 (DAO Layer):
    • Gọi ProductDAO.update() cập nhật các thông tin mô tả chung của sản phẩm trong bảng products. (Xem code Java nhãn [Bước 3 - Cập nhật thông tin chung sản phẩm])
    • Tìm các phân loại cũ bị xóa trên giao diện để gọi ProductVariantDAO.deactivate() (soft-delete). (Xem code Java nhãn [Bước 3 - Soft-delete biến thể bị bỏ])
    • Duyệt danh sách biến thể gửi lên: Nếu là biến thể mới (chưa có ID) thì gọi ProductVariantDAO.save(), nếu là biến thể cũ thì gọi ProductVariantDAO.update(). (Xem code Java nhãn [Bước 3 - Cập nhật hoặc lưu biến thể mới/cũ])
    • Đồng bộ danh sách bao bì đóng gói: cập nhật, lưu mới và xóa các bao bì đóng gói không còn giữ lại. (Xem code Java nhãn [Bước 3 - Cập nhật danh sách bao bì đóng gói])
  • Bước 4 (Database & Redirect): Cập nhật dữ liệu vào cơ sở dữ liệu, ghi nhận thay đổi, lưu thông điệp thành công vào HTTP Session và redirect người dùng (PRG Pattern) về danh sách sản phẩm. (Xem code Java nhãn [Bước 4 - Trả về phản hồi và redirect])

// ProductEditServlet.java - Phương thức doPost() cập nhật thông tin sản phẩm và các biến thể phân loại cũ/mới
@Override
protected void doPost(HttpServletRequest req, HttpServletResponse resp)
        throws ServletException, IOException {
    req.setCharacterEncoding("UTF-8");
    resp.setCharacterEncoding("UTF-8");

    // [Bước 2 - Auth & Owner Check]: Xác thực quyền chủ shop và kiểm tra sự tồn tại của sản phẩm
    HttpSession session = req.getSession();
    User currentUser = SessionUtil.getCurrentUser(session);
    if (currentUser == null || !AppConfig.ROLE_SHOP_OWNER.equals(currentUser.getRole())) {
        resp.sendRedirect(req.getContextPath() + "/auth/login");
        return;
    }

    int productId = Integer.parseInt(req.getParameter("id").trim());
    try {
        Product p = productDAO.findOneById(productId);
        if (p == null || p.getOwnerId() != currentUser.getUserId() || "DELETED".equals(p.getStatus())) {
            resp.sendError(HttpServletResponse.SC_FORBIDDEN, "Bạn không có quyền chỉnh sửa sản phẩm này.");
            return;
        }

        // [Bước 2 - Đọc tham số cập nhật]: Đọc các trường thuộc tính mới
        String name = req.getParameter("name");
        String description = req.getParameter("description");
        String originCountry = req.getParameter("originCountry");
        String originRegion = req.getParameter("originRegion");
        String harvestDateStr = req.getParameter("harvestDate");
        String shelfLifeStr = req.getParameter("shelfLifeDays");
        String storageInstruction = req.getParameter("storageInstruction");
        String categoryIdStr = req.getParameter("categoryId");
        String status = req.getParameter("status");
        boolean isOrganic = req.getParameter("isOrganic") != null;
        boolean isImported = req.getParameter("isImported") != null;

        String[] variantIds = req.getParameterValues("variantId");
        String[] variantLabels = req.getParameterValues("variantLabel");
        String[] variantPrices = req.getParameterValues("variantPrice");
        String[] variantWeights = req.getParameterValues("variantWeight");
        String[] variantDiscountPrices = req.getParameterValues("variantDiscountPrice");
        String[] variantDiscountStarts = req.getParameterValues("variantDiscountStart");
        String[] variantDiscountEnds = req.getParameterValues("variantDiscountEnd");

        String[] packagingIds = req.getParameterValues("packagingId");
        String[] packagingLabels = req.getParameterValues("packagingLabel");
        String[] packagingPriceAdds = req.getParameterValues("packagingPriceAdd");

        // [Bước 2 - Validate dữ liệu cập nhật]: Thực hiện validate thông tin cập nhật
        List errors = new ArrayList<>();
        if (name == null || name.trim().isEmpty()) {
            errors.add("Tên sản phẩm không được để trống.");
        }
        if (variantLabels == null || variantLabels.length == 0) {
            errors.add("Sản phẩm phải có ít nhất một phân loại/biến thể.");
        }

        if (!errors.isEmpty()) {
            SessionUtil.flashError(session, String.join("
", errors));
            resp.sendRedirect(req.getContextPath() + "/shop/products");
            return;
        }

        // [Bước 3 - Cập nhật thông tin chung sản phẩm]: Lưu lại cập nhật sản phẩm vào DB
        p.setName(name.trim());
        p.setDescription(description != null ? description.trim() : null);
        p.setCategoryId(Integer.parseInt(categoryIdStr));
        p.setOriginCountry(originCountry != null ? originCountry.trim() : null);
        p.setOriginRegion(originRegion != null ? originRegion.trim() : null);
        p.setHarvestDate(harvestDateStr != null && !harvestDateStr.trim().isEmpty() ? LocalDate.parse(harvestDateStr) : null);
        p.setShelfLifeDays(shelfLifeStr != null && !shelfLifeStr.trim().isEmpty() ? Integer.parseInt(shelfLifeStr) : null);
        p.setStorageInstruction(storageInstruction != null ? storageInstruction.trim() : null);
        p.setStatus(status);
        p.setIsOrganic(isOrganic);
        p.setIsImported(isImported);

        productDAO.update(p);

        List existingVariants = productVariantDAO.findByProduct(productId);
        Map existingVariantMap = new java.util.LinkedHashMap<>();
        if (existingVariants != null) {
            for (ProductVariant ev : existingVariants) {
                existingVariantMap.put(ev.getVariantId(), ev);
            }
        }

        Set submittedIds = new HashSet<>();
        if (variantIds != null) {
            for (String vidStr : variantIds) {
                if (vidStr != null && !vidStr.trim().isEmpty()) {
                    submittedIds.add(Integer.parseInt(vidStr.trim()));
                }
            }
        }

// ProductDAO.java - Xóa mềm sản phẩm và xử lý giỏ hàng/biến thể liên quan trong Transaction
public void deleteProduct(int productId) throws SQLException {
    String updateProductSql = "UPDATE products SET status = 'DELETED', updated_at = GETDATE() WHERE product_id = ?";
    String updateVariantsSql = "UPDATE product_variants SET is_active = 0, updated_at = GETDATE() WHERE product_id = ?";
    String deleteCartSql = "DELETE FROM cart_items WHERE variant_id IN (SELECT variant_id FROM product_variants WHERE product_id = ?)";
    String updatePromotionsSql = "UPDATE promotions SET is_active = 0, updated_at = GETDATE() WHERE product_id = ?";
    
    // [Bước 3 - Bắt đầu Transaction và xóa mềm sản phẩm]: Khởi tạo Transaction bằng cách set AutoCommit = false
    try (Connection conn = getConnection()) {
        conn.setAutoCommit(false);
        try (PreparedStatement psCart = conn.prepareStatement(deleteCartSql);
             PreparedStatement psVar = conn.prepareStatement(updateVariantsSql);
             PreparedStatement psProm = conn.prepareStatement(updatePromotionsSql);
             PreparedStatement psProd = conn.prepareStatement(updateProductSql)) {
            
            // [Bước 3 - Xóa sản phẩm khỏi giỏ hàng]: Xóa sản phẩm khỏi giỏ hàng của tất cả khách hàng
            psCart.setInt(1, productId);
            psCart.executeUpdate();
            
            // [Bước 3 - Hủy kích hoạt biến thể]: Tắt hoạt động toàn bộ biến thể sản phẩm
            psVar.setInt(1, productId);
            psVar.executeUpdate();
            
            // [Bước 3 - Hủy khuyến mãi liên quan]: Tắt hoạt động các khuyến mãi áp dụng riêng cho sản phẩm này
            psProm.setInt(1, productId);
            psProm.executeUpdate();
            
            // [Bước 3 - Cập nhật status sản phẩm]: Cập nhật status sản phẩm thành DELETED
            psProd.setInt(1, productId);
            psProd.executeUpdate();
            
            // [Bước 3 - Commit Transaction]: Xác nhận các thay đổi dữ liệu thành công
            conn.commit();
        } catch (SQLException e) {
            conn.rollback();
            throw e;
        }
    }
}

// InventoryService.java - Kiểm tra tồn kho sau thay đổi và kích hoạt cảnh báo in-app
private void checkAndSendLowStockAlert(Connection conn, int variantId, int stockAfter) {
    try {
        int ownerId = productVariantDAO.getProductOwnerId(conn, variantId);
        int threshold = 5; // Mặc định là 5
        // [Bước 2 - Truy vấn ngưỡng cảnh báo tồn kho của Shop]: Đọc low_stock_threshold từ shop_owner_profiles
        String thresholdSql = "SELECT low_stock_threshold FROM shop_owner_profiles WHERE user_id = ?";
        try (PreparedStatement ps = conn.prepareStatement(thresholdSql)) {
            ps.setInt(1, ownerId);
            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    threshold = rs.getInt("low_stock_threshold");
                }
            }
        }

        // [Bước 3 - Gửi thông báo cảnh báo tồn kho thấp]: Nếu lượng tồn kho giảm xuống dưới hoặc bằng ngưỡng cấu hình
        if (stockAfter <= threshold) {
            Map info = productVariantDAO.getVariantAndProductName(conn, variantId);
            String sku = (String) info.get("sku");
            String productName = (String) info.get("name");
            
            String title = "Cảnh báo tồn kho thấp";
            String message = "Sản phẩm \"" + productName + "\" (SKU: " + sku + ") sắp hết hàng. Chỉ còn " + stockAfter + " sản phẩm.";
            notificationService.send(ownerId, AppConfig.NOTIF_INVENTORY_ALERT, title, message, "/shop/products");
        }
    } catch (Exception ex) {
        LoggerUtil.warn(log, "Không thể gửi cảnh báo tồn kho thấp", ex);
    }
}

@Override
protected void doPost(HttpServletRequest req, HttpServletResponse resp)

    HttpSession session = req.getSession(false);

    if (user == null) {
        resp.setStatus(HttpServletResponse.SC_UNAUTHORIZED);

        return;
    }

    // Xác thực tính hợp lệ của CSRF Token chống tấn công giả mạo yêu cầu
    String sessionToken = session != null ? (String) session.getAttribute(AppConfig.SESSION_CSRF_TOKEN) : null;
    String requestToken = req.getHeader("X-CSRF-Token");
    if (requestToken == null || requestToken.trim().isEmpty()) {
        requestToken = req.getParameter("_csrf");
    }
    if (sessionToken == null || !sessionToken.equals(requestToken)) {
        resp.setStatus(HttpServletResponse.SC_FORBIDDEN);
        JsonUtil.writeJson(resp, ApiResponse.fail(403, "CSRF token không hợp lệ."));
        return;
    }

    String action = req.getParameter("action");
    try {
        if ("markRead".equals(action)) {
            // [Bước 2 - Đánh dấu đã đọc cho một thông báo cụ thể]: Đánh dấu đã đọc cho một thông báo cụ thể

            resp.setStatus(HttpServletResponse.SC_BAD_REQUEST);
            JsonUtil.writeJson(resp, ApiResponse.fail(400, "Hành động không hợp lệ."));
        }

        );
        if (ownerId != null) {

        }
        sql.append("AND p.category_id = ? ");
        params.add(categoryId);
        sql.append("GROUP BY o.cancellation_reason ORDER BY cancel_count DESC");
    } else {

            "SELECT COALESCE(cancellation_reason, N'Không có lý do') AS reason, COUNT(*) AS cancel_count " +

        );
        if (ownerId != null) {

        if (startDate != null && !startDate.trim().isEmpty()) {
            sql.append("AND CAST(created_at AS DATE) >= ? ");
            params.add(java.sql.Date.valueOf(startDate));
        }
        if (endDate != null && !endDate.trim().isEmpty()) {
            sql.append("AND CAST(created_at AS DATE) <= ? ");
            params.add(java.sql.Date.valueOf(endDate));
        }
        sql.append("GROUP BY cancellation_reason ORDER BY cancel_count DESC");
    }

    // [Bước 2 - Thống kê lý do hủy đơn hàng của Shop]: Thực hiện truy vấn các lý do hủy đơn và số lượng tương ứng từ DB
    try (Connection conn = getConnection();
         PreparedStatement ps = conn.prepareStatement(sql.toString())) {
        for (int i = 0; i < params.size(); i++) {
            ps.setObject(i + 1, params.get(i));
        }
        try (ResultSet rs = ps.executeQuery()) {
            while (rs.next()) {
                Map map = new HashMap<>();
                map.put("reason", rs.getString("reason"));
                map.put("count", rs.getInt("cancel_count"));
                list.add(map);
            }
        }
    }
    return list;

Chức năng 15: Tiến trình chạy ngầm tự động đối soát và quyết toán (Auto-Settlement)

1. Thông tin chung

  • Tên chức năng: Tiến trình chạy ngầm quyết toán tự động đơn hàng đã hoàn thành.
  • Đối tượng sử dụng (Actor): Hệ thống tự động chạy ngầm (System Background Job).
  • Mục tiêu: Tự động kết chuyển dòng tiền của đơn giao thành công (sau 12 tiếng đóng băng đổi trả) sang ví khả dụng của Shop Owner mà không cần duyệt thủ công.

2. Luồng hoạt động chi tiết

  • Khởi tạo (`AutoSettlementListener`): Khi ứng dụng khởi chạy trên Tomcat Server, sự kiện contextInitialized kích hoạt. Bộ lập lịch ScheduledExecutorService được đăng ký và chạy dưới một luồng độc lập AutoSettlement-Thread định kỳ. (Xem code Java nhãn [Bước 1 - Lập lịch executor chạy định kỳ hourly job]).
  • Quá trình thực hiện (Chạy định kỳ mỗi 1 tiếng):
    • Đọc cấu hình thời gian đóng băng đơn hàng an toàn (ví dụ: 12 tiếng) từ bảng cấu hình.
    • Quét toàn bộ đơn hàng ở trạng thái DELIVERED có thời gian hoàn thành vượt quá 12 tiếng và chưa từng được đối soát quyết toán.
    • Quyết toán trong Transaction: Tạo bản ghi đối soát mới (ShopSettlement), lưu vết liên kết đơn hàng đối soát (ShopSettlementOrder) và thực hiện cộng doanh thu thực tế (sau trừ phí sàn) vào ví khả dụng của Shop Owner tương ứng. (Xem code Java nhãn [Bước 2 - Tạo kỳ đối soát và lưu các đơn hàng liên quan vào Database Transaction]).
  • Hủy tiến trình: Khi server Tomcat dừng, sự kiện contextDestroyed tắt bộ lập lịch ngầm an toàn để giải phóng tài nguyên.

// 1. AutoSettlementListener.java - Lập lịch thread chạy ngầm định kỳ 1 tiếng một lần
@Override
public void contextInitialized(ServletContextEvent sce) {
    LoggerUtil.info(log, "[AutoSettlementListener] Khởi tạo daily background job cho quyết toán tự động...");
    scheduler = Executors.newSingleThreadScheduledExecutor(runnable -> {
        Thread t = new Thread(runnable, "AutoSettlement-Thread");
        t.setDaemon(true);
        return t;
    });

    // [Bước 1 - Lập lịch executor chạy định kỳ hourly job]: Đăng ký và lập lịch executor chạy định kỳ hourly job chốt đối soát cho shop
    scheduler.scheduleWithFixedDelay(() -> {
        try {
            LoggerUtil.info(log, "[AutoSettlementListener] Bắt đầu tiến trình quyết toán đơn hàng tự động...");
            int processed = settlementService.runAutoSettlement();
            LoggerUtil.info(log, "[AutoSettlementListener] Hoàn thành tiến trình quyết toán. Đã tạo " + processed + " kỳ đối soát mới.");
        } catch (Exception e) {
            LoggerUtil.error(log, "[AutoSettlementListener] Lỗi khi chạy job quyết toán tự động", e);
        }
    }, 10, 60 * 60, TimeUnit.SECONDS);
}

// 2. SettlementDAO.java - Thực hiện quyết toán đơn hàng tự động trong Database Transaction
private int runAutoSettlementInternal(int freezeValue, String datepart) throws SQLException {
    int settlementsCreated = 0;
    List<Integer> owners = new ArrayList<>();
    
    // Tìm các Shop Owner có đơn hàng thỏa mãn điều kiện quyết toán (DELIVERED quá 12h, không bị hoàn trả, chưa đối soát)
    String getOwnersSql = "SELECT DISTINCT o.owner_id FROM orders o "
                        + "LEFT JOIN deliveries d ON d.order_id = o.order_id "
                        + "WHERE o.status = 'DELIVERED' "
                        + "  AND NOT EXISTS (SELECT 1 FROM shop_settlement_orders sso WHERE sso.order_id = o.order_id) "
                        + "  AND NOT EXISTS (SELECT 1 FROM return_requests r WHERE r.order_id = o.order_id AND r.status IN ('REQUESTED', 'PROCESSING', 'APPROVED')) "
                        + "  AND COALESCE(d.delivered_at, o.updated_at) < DATEADD(" + datepart + ", ?, GETDATE())";
                        
    try (Connection conn = getConnection()) {
        try (PreparedStatement ps = conn.prepareStatement(getOwnersSql)) {
            ps.setInt(1, -freezeValue);
            try (ResultSet rs = ps.executeQuery()) {
                while (rs.next()) {
                    owners.add(rs.getInt("owner_id"));
                }
            }
        }

        if (owners.isEmpty()) return 0;

        // [Bước 2 - Tạo kỳ đối soát và lưu các đơn hàng liên quan vào Database Transaction]: Khởi tạo Transaction bằng cách set AutoCommit = false
        conn.setAutoCommit(false);
        try {
            String getOrdersSql = "SELECT o.order_id, o.final_amount, o.platform_fee, o.discount_amount, o.created_at, "
                                + "  COALESCE((SELECT SUM(r.refund_amount) FROM return_requests r WHERE r.order_id = o.order_id AND r.status = 'COMPLETED'), 0) AS refund_amount "
                                + "FROM orders o LEFT JOIN deliveries d ON d.order_id = o.order_id "
                                + "WHERE o.owner_id = ? AND o.status = 'DELIVERED' "
                                + "  AND NOT EXISTS (SELECT 1 FROM shop_settlement_orders sso WHERE sso.order_id = o.order_id) "
                                + "  AND NOT EXISTS (SELECT 1 FROM return_requests r WHERE r.order_id = o.order_id AND r.status IN ('REQUESTED', 'PROCESSING', 'APPROVED')) "
                                + "  AND COALESCE(d.delivered_at, o.updated_at) < DATEADD(" + datepart + ", ?, GETDATE())";
                                
            String insertSettlementSql = "INSERT INTO shop_settlements (owner_id, period_start, period_end, gross_amount, platform_fee_amount, refund_amount, adjustment_amount, net_amount, status, calculated_at, created_by, note) "
                                       + "VALUES (?, ?, ?, ?, ?, ?, 0, ?, 'PENDING', GETDATE(), 1, ?)";
                                       
            String insertSettlementOrderSql = "INSERT INTO shop_settlement_orders (settlement_id, order_id, order_amount, platform_fee_amount, discount_amount, refund_amount, net_amount) "
                                            + "VALUES (?, ?, ?, ?, ?, ?, ?)";
                                            
            try (PreparedStatement psOrders = conn.prepareStatement(getOrdersSql);
                 PreparedStatement psInsertSettle = conn.prepareStatement(insertSettlementSql, Statement.RETURN_GENERATED_KEYS);
                 PreparedStatement psInsertSettleOrder = conn.prepareStatement(insertSettlementOrderSql)) {
                 
                for (int ownerId : owners) {
                    psOrders.setInt(1, ownerId);
                    psOrders.setInt(2, -freezeValue);
                    
                    List<Map<String, Object>> ordersList = new ArrayList<>();
                    java.sql.Date periodStart = null;
                    java.sql.Date periodEnd = null;
                    BigDecimal gross = BigDecimal.ZERO;
                    BigDecimal platformFee = BigDecimal.ZERO;
                    BigDecimal refund = BigDecimal.ZERO;
                    
                    try (ResultSet rsOrders = psOrders.executeQuery()) {
                        while (rsOrders.next()) {
                            Map<String, Object> orderData = new HashMap<>();
                            int orderId = rsOrders.getInt("order_id");
                            BigDecimal finalAmount = rsOrders.getBigDecimal("final_amount");
                            BigDecimal pFee = rsOrders.getBigDecimal("platform_fee");
                            BigDecimal disc = rsOrders.getBigDecimal("discount_amount");
                            BigDecimal ref = rsOrders.getBigDecimal("refund_amount");
                            java.sql.Date orderDate = rsOrders.getDate("created_at");
                            
                            orderData.put("order_id", orderId);
                            orderData.put("final_amount", finalAmount);
                            orderData.put("platform_fee", pFee);
                            orderData.put("discount_amount", disc);
                            orderData.put("refund_amount", ref);
                            ordersList.add(orderData);
                            
                            gross = gross.add(finalAmount != null ? finalAmount : BigDecimal.ZERO);
                            platformFee = platformFee.add(pFee != null ? pFee : BigDecimal.ZERO);
                            refund = refund.add(ref != null ? ref : BigDecimal.ZERO);
                            
                            if (orderDate != null) {
                                if (periodStart == null || orderDate.before(periodStart)) periodStart = orderDate;
                                if (periodEnd == null || orderDate.after(periodEnd)) periodEnd = orderDate;
                            }
                        }
                    }
                    
                    if (ordersList.isEmpty()) continue;
                    if (periodStart == null) periodStart = new java.sql.Date(System.currentTimeMillis());
                    if (periodEnd == null) periodEnd = new java.sql.Date(System.currentTimeMillis());
                    
                    // Doanh thu thực nhận sau khi trừ phí sàn 5% và hoàn trả
                    BigDecimal net = gross.subtract(platformFee).subtract(refund);
                    String note = String.format("Quyết toán tự động hệ thống cho %d đơn hàng giao thành công.", ordersList.size());
                    
                    // Tạo bản ghi chốt đối soát shop_settlement
                    psInsertSettle.setInt(1, ownerId);
                    psInsertSettle.setDate(2, periodStart);
                    psInsertSettle.setDate(3, periodEnd);
                    psInsertSettle.setBigDecimal(4, gross);
                    psInsertSettle.setBigDecimal(5, platformFee);
                    psInsertSettle.setBigDecimal(6, refund);
                    psInsertSettle.setBigDecimal(7, net);
                    psInsertSettle.setString(8, note);
                    psInsertSettle.executeUpdate();
                    
                    int settlementId = -1;
                    try (ResultSet gk = psInsertSettle.getGeneratedKeys()) {
                        if (gk.next()) settlementId = gk.getInt(1);
                    }
                    
                    // Thêm liên kết các đơn hàng đối soát chi tiết vào bảng shop_settlement_orders
                    for (Map<String, Object> orderData : ordersList) {
                        int orderId = (int) orderData.get("order_id");
                        BigDecimal finalAmount = (BigDecimal) orderData.get("final_amount");
                        BigDecimal pFee = (BigDecimal) orderData.get("platform_fee");
                        BigDecimal disc = (BigDecimal) orderData.get("discount_amount");
                        BigDecimal ref = (BigDecimal) orderData.get("refund_amount");
                        BigDecimal netOrder = finalAmount.subtract(pFee).subtract(ref);
                        
                        psInsertSettleOrder.setInt(1, settlementId);
                        psInsertSettleOrder.setInt(2, orderId);
                        psInsertSettleOrder.setBigDecimal(3, finalAmount);
                        psInsertSettleOrder.setBigDecimal(4, pFee);
                        psInsertSettleOrder.setBigDecimal(5, disc);
                        psInsertSettleOrder.setBigDecimal(6, ref);
                        psInsertSettleOrder.setBigDecimal(7, netOrder);
                        psInsertSettleOrder.addBatch();
                    }
                    psInsertSettleOrder.executeBatch();
                    settlementsCreated++;
                    
                    // Bắn thông báo In-app cho Shop Owner biết đã chốt kỳ đối soát mới
                    try {
                        service.chat.NotificationService notifService = new service.chat.NotificationService();
                        notifService.send(ownerId, "PAYMENT", "Đối soát tự động mới", 
                            String.format("Hệ thống đã tự động chốt kỳ đối soát mới #%d với doanh thu gộp: %,.0f đ, thực nhận: %,.0f đ. Vui lòng kiểm tra chi tiết.", 
                            settlementId, gross.doubleValue(), net.doubleValue()), "/shop/settlement");
                    } catch (Exception e) {
                        LoggerUtil.warn(log, "Failed to send auto-settlement notification to shop owner " + ownerId, e);
                    }
                }
            }
            conn.commit();
        } catch (SQLException e) {
            conn.rollback();
            throw e;
        }
    }
    return settlementsCreated;
